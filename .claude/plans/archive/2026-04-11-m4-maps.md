# Milestone 4: Interactive Maps (Atlas)

**Date:** 2026-04-11

---

## Goal

Upload map images, display with pan/zoom, place pins linked to wiki pages, and navigate via the Atlas sidebar. Core map functionality — nesting, layers, and tiling deferred.

**References:**
- Design spec: `design/specs/milestone-4-maps.md`

## Chosen Approach

Store map images using the existing images blob table. Map metadata and pins in new tables. Canvas-based viewer with pan/zoom. Pins as overlaid DOM elements for easy interaction. Atlas nav in sidebar opens map list/viewer.
